1.更换主题

先到主题网站去看喜欢哪一个模板，比如我喜欢的是Chic主题，然后右键复制链接，老样子shift打开shell，先把模板下载到本地；

git clone https://siricee.github.io/hexo-theme-Chic/ themes/Chic
# git clone 模板链接 指定保存位置

2.站点具体配置

打开博客目录，找到配置文件，拉到最底下，修改成指定的主题模板；

然后拉到最上面，找到Site，可以设置title等…



再找到下面这个，
source文件就是放各种源文件的，比如你新建的博客；
public就是生成的静态文件，命令hexo g可生成，只有在public中生成，才能在你的博客中出现；
archives文件是你的发出去的博客归档集合，是一个排版；
**重点来了！！！！！**
tag和category文件是标签和目录，
但这两个文件需要自己先创建，在创建之前，先打开Chic主题的排版文件，
可以发现对应的tag和category，换其他主题的时候可能是tags或者categories，总而言之，不同主题，就看它们的layout里面，tag和category是什么形式，这里都是单数。
然后我们shift+shell，创建目录,自动在source文件中， 
hexo new page tag
hexo new page category
